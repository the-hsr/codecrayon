# CodeCrayon

A customizable code snippet generator built for creating beautiful, shareable code images with minimal effort.

---

## Features

- Multiple language support (JavaScript, TypeScript, Python, Swift)
- Syntax highlighting using PrismJS
- macOS and Windows style code windows
- Adjustable padding and layout
- Line numbers toggle
- Gradient background themes
- Resizable editor width
- Export code as high-resolution PNG

---

## Preview

> Generate clean, aesthetic code snippets ready for sharing on social media or documentation.

---

## Tech Stack

- React (Next.js App Router)
- TypeScript
- Tailwind CSS
- PrismJS
- html-to-image
- react-simple-code-editor
